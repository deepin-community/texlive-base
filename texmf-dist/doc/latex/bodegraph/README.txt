
bodegraph (bode graph)

version : 1.6

Ce package permet de tracer les diagrammes de Bode, Black et Nyquist à laide de 
Gnuplot et Tikz. Les fonctions de transfert élémentaires et les correcteurs courants 
sont préprogrammés pour être utilisés dans les fonctions de tracé.

Le fichier zip comporte:
- bodegraph.sty : le package
- isoa.txt : macro-commandes de définition des courbes iso-module;
- isoam.txt : macro-commandes de définition des courbes iso-argument;

- bodegraph.tex : le source de la documentation
- bodegraphe.pdf :
- /gnuplot/bodegraph : le répertoire comportant les fichiers Gnumplot pré-calculés.

This package allows you to draw the Bode plots, Nyquist, and Black using Gnuplot 
and Tikz. Elementary Functions Transfer and basics correctors are preprogrammed
to be used.

The zip file includes:
- bodegraph.sty: the package
- isoa.txt: macro-commands for defining iso-module curves;
- isoam.txt: macro-commands for defining iso-argument curves;

- bodegraph.tex: the source of the documentation
- bodegraphe.pdf:
- /gnuplot/bodegraph: the directory containing the pre-calculated Gnuplot files.
